# VIBECODE2
enscapulation code project
The first thing it will do is count the people entering the supermarket and observe what they purchase; in this way, supermarkets can maintain better track of what interests their customers.

The second feature is an option that allows for the digital adjustment of product prices, eliminating the need for a person to manually swap out paper price tags.

The third feature is the ability to apply discounts and special offers directly to the products themselves; this ensures that when a shopper picks up an item, they can immediately see the final price, including the applied discount or offer.
